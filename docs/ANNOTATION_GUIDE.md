# 标注系统使用指南

## 功能概述

标注系统是一个功能完善的屏幕标注工具，允许用户在截图上进行各种标注操作。系统提供直观的颜色选择、多种标注工具和便捷的编辑功能。

## 主要功能

### 1. 标注工具

系统提供 5 种标注工具：

- **矩形工具** 📐：绘制矩形框，适合圈出重点区域
- **圆形工具** ⭕：绘制椭圆形，用于标记特定对象
- **箭头工具** ➡️：绘制带箭头的直线，用于指示方向
- **自由绘制工具** ✏️：自由手绘，适合个性化标记
- **文字工具** 📝：添加文字说明，提供详细注释

### 2. 颜色选择

提供 5 种常用颜色，每种颜色都有直观的圆形色块标识：

- 🔴 **红色** (#ef4444) - 醒目，用于重要标记
- 🔵 **蓝色** (#3b82f6) - 清晰，用于一般标记
- 🟢 **绿色** (#10b981) - 柔和，用于确认标记
- 🟡 **黄色** (#f59e0b) - 明亮，用于高亮标记
- 🟣 **紫色** (#8b5cf6) - 独特，用于特殊标记

### 3. 线条粗细

提供 5 种线条粗细选项（2px、4px、6px、8px、10px），可根据需要选择：
- 细线条（2-4px）：适合精细标注
- 中等线条（6px）：适合一般标注
- 粗线条（8-10px）：适合强调标记

### 4. 编辑功能

- **选择工具**：选择和移动已创建的标注
- **撤销/重做**：支持 Ctrl+Z 撤销，Ctrl+Y 重做
- **删除标注**：
  - 选中后按 Delete/Backspace 键
  - 点击标注列表中的删除按钮
- **清除所有**：一键清除所有标注

### 5. 标注列表

右侧显示标注列表，包含：
- 颜色指示器：显示标注使用的颜色
- 类型标识：显示标注类型（矩形、圆形等）
- 删除按钮：快速删除单个标注
- 点击可选择标注进行编辑

## 使用方法

### 进入标注模式

1. 在截图查看界面，点击底部工具栏的 **标注按钮**（编辑图标）
2. 系统会自动加载之前保存的标注（如果有）
3. 进入标注编辑界面

### 创建标注

1. **选择工具**：点击工具栏中的标注工具图标
2. **选择颜色**：点击颜色选择器中的颜色
3. **选择粗细**：点击线条粗细选项
4. **绘制标注**：
   - 矩形/圆形/箭头：在图片上按住鼠标左键拖动
   - 自由绘制：按住鼠标左键自由绘制
   - 文字：点击位置，输入文字后按 Enter

### 编辑标注

1. 切换到 **选择工具**（光标图标）
2. 点击要编辑的标注
3. 拖动标注到新位置
4. 按 Delete 键可删除

### 撤销和重做

- **撤销**：点击撤销按钮 或 按 Ctrl+Z
- **重做**：点击重做按钮 或 按 Ctrl+Y

### 保存标注

- 点击 **完成按钮**（✓）自动保存标注
- 标注数据会自动保存到本地存储
- 下次打开同一图片时会自动加载之前的标注

### 取消标注

- 点击 **取消按钮**（✕）放弃本次标注
- 如果有已保存的标注，会恢复到之前的状态

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| Delete/Backspace | 删除选中的标注 |
| Enter | 确认文字输入 |
| Escape | 取消文字输入/取消选择 |

## 数据存储

- 标注数据保存在浏览器的 localStorage 中
- 每个图片生成唯一的 ID 进行标识
- 最多保存最近 50 条标注记录
- 标注数据包括：标注类型、位置、颜色、粗细等

## 最佳实践

1. **颜色搭配**：
   - 重要内容使用红色
   - 一般标记使用蓝色
   - 确认信息使用绿色
   - 高亮显示使用黄色

2. **标注层次**：
   - 先绘制大区域（矩形/圆形）
   - 再添加指示（箭头）
   - 最后添加说明（文字）

3. **性能优化**：
   - 避免创建过多标注（建议不超过 50 个）
   - 定期清理不需要的标注

## 常见问题

### Q: 标注保存后在哪里查看？
A: 标注会自动保存在本地，下次打开同一图片时会自动加载。

### Q: 可以导出标注数据吗？
A: 目前标注会直接绘制到图片上，保存图片即可包含标注内容。

### Q: 标注会占用多少存储空间？
A: 每条标注数据很小（约 1-5KB），50 条标注约占用 250KB。

### Q: 可以修改已保存的标注吗？
A: 可以，重新进入标注模式后，可以编辑或删除已有标注。

## 技术细节

### 标注数据结构

```javascript
{
  id: 1234567890,          // 唯一标识
  type: 'rect',            // 标注类型
  color: '#ef4444',        // 颜色值
  strokeWidth: 4,          // 线条粗细
  x: 100,                  // X 坐标
  y: 100,                  // Y 坐标
  width: 200,              // 宽度
  height: 150              // 高度
}
```

### 支持的标注类型

- `rect`: 矩形
- `circle`: 圆形
- `arrow`: 箭头
- `free`: 自由绘制
- `text`: 文字
